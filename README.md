# MONTE PROJECT - Mars Orbital Network Terraform Exploration

[![Status](https://img.shields.io/badge/Status-Active-success.svg)]()
[![Platform](https://img.shields.io/badge/Platform-Web-blue.svg)]()

A premium, SpaceX-inspired multi-page website dedicated to the future of Mars colonization and terraforming.

## 🚀 Kluczowe Sekcje / Key Sections

- **Mission (Misja):** Strategic goals for long-term human survival.
- **Technology (Technologia):** Heavy-lift rockets, fusion engines, and life support.
- **Colonization (Kolonizacja):** Phases of building self-sufficient Martian cities.
- **Resources (Zasoby):** Infrastructure for water ice mining and oxygen production.
- **Hazards (Zagrożenia):** In-depth analysis of radiation, dust storms, and psychological challenges.
- **Timeline (Oś Czasu):** Interactive roadmap from first tests to a thriving colony.
- **Kardashev Scale:** Interactive module visualizing the evolution into a Galatic Civilization.

## 🛠️ Stack Technologiczny / Tech Stack

- **HTML5:** Semantic architecture for SEO and accessibility.
- **CSS3:** Custom animations, responsive layouts, and SpaceX-inspired "Dark Mode" aesthetic.
- **JavaScript:** Interactive UI components, timeline engine, and dynamic background transitions.
- **Media:** High-quality video backgrounds and custom-designed visual assets.

## 📦 Lokalna Instalacja / Local Setup

1. Sklonuj repozytorium / Clone the repository:
   ```bash
   git clone https://github.com/your-username/mars-monte-project.git
   ```
2. Otwórz `index.html` w przeglądarce / Open `index.html` in your browser.
